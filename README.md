# healthcare-analytics-portfolio
Healthcare analytics portfolio showcasing financial performance and labor efficiency analysis using Excel. Includes KPI development, PivotTables, and data-driven insights aligned with healthcare operations and FP&amp;A.

This portfolio contains healthcare-focused analytics projects demonstrating financial analysis, labor efficiency evaluation, and data-driven decision-making using Excel.

Projects:

1. Kaiser Operating Performance Analysis
- Analyzed revenue, expenses, and operating margin trends from 2019–2023
- Identified margin compression due to cost growth exceeding revenue
- Built PivotTables, KPIs, and visualizations

2. Healthcare Labor Efficiency Analysis
- Evaluated labor efficiency across departments using hours and cost per encounter
- Identified ICU as the most resource-intensive department
- Developed comparative analysis using PivotTables and charts
